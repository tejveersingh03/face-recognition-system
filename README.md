# Face Recognition System

## Description
This project is a Face Recognition System built using Python and OpenCV. It detects and recognizes human faces in real-time using a webcam.

## Features
- Real-time face detection
- Face recognition using trained data
- Uses webcam for capturing images
- Simple and efficient system

## Technologies Used
- Python
- OpenCV
- NumPy

## How It Works
1. Capture face images and store them
2. Train the model using face data
3. Run the recognition system
4. Detect and recognize faces through webcam

## Installation
1. Install Python
2. Install required libraries:
   pip install opencv-python
   pip install numpy

## Usage
Run the main Python file:
python main.py
